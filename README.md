Meu foco vai muito além de construir apenas um portfólio técnico: sou dedicada a desenvolver soluções inteligentes, APIs robustas e automações que transformam dados em eficiência e mudam dinâmicas de negócios diariamente.

Minhas atividades unem a engenharia de software com a estratégia de dados para otimizar sistemas e trazer eficiência para o ecossistema digital.
